# 🚀 Linux Package & Service Automation Tool

A practical Bash automation script that simplifies common Linux system administration tasks.

This tool automates:

- ✅ System update
- ✅ System upgrade
- ✅ Package installation
- ✅ Service status check

It demonstrates how shell scripting can streamline DevOps workflows and eliminate repetitive manual commands.

---

## 📌 Overview

Managing packages and checking services manually requires multiple terminal commands.  
This script consolidates them into one interactive workflow.

Instead of running:

- `sudo apt update`
- `sudo apt upgrade`
- `sudo apt install package`
- `sudo systemctl status service`

You can automate everything with a single script.
